# TestTool
常用小工具集合（暂时只有一个）

1.爱心锁
作为程序员怎么能不注意休息呢,为了关爱你们的健康，现推出一款winfrom程序
每60分钟自动锁屏一次（弹窗全屏）,锁屏时间为3分钟，关爱程序员健康发展，从我做起


使用说明 
1.双击启动应用程序，在任务管理器>进程里面会有 LoveLock.exe  不用了可以结束进程（注意有开机自启动，不需要删除源文件就可以了）
2.轮播图片直接添加到imgs文件夹就可以了，注意分辨率
